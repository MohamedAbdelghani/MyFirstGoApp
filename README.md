# Routes
### `POST /AddPerson`
adds new person

### `PUT /EditPerson`
Updates person

### `DELETE /DeletePerson`
Removes person

### `GET /GetById/{id}`
Retrieves person of specified id.

### `GET /GetByWeight/{weight}`
Retrieves persons who have specified weight.

### `GET /GetByHeight/{height}`
Retrieves persons who have specified height.
